Measuring interactions

Measure and time any interaction on a page.

Code

    // mark the starting point
    performance.mark('actionStart');

    // measure time from start
    performance.measure('actionTime', 'actionStart');

    // retrieve timing data;
    var perfEntries = performance.getEntriesByType("measure");
    for(i = 0; i < perfEntries.length; i++){
      console.log(perfEntries[i].name + ' took ' + perfEntries[i].duration.toFixed(2) + ' ms');
    }
